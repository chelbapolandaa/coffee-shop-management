# Coffee Shop Management (Laravel)

Project ini adalah **website cafe online** lengkap dengan:

- Menu publik
- Promo
- Keranjang / checkout online
- Admin panel untuk:
  - Manajemen menu
  - Pesanan
  - Promo
  - Struktur organisasi
  - Laporan
- Database dummy untuk struktur organisasi sudah tersedia
